Name

Serial.begin - Set speed for the integrated UART ( serial port ).

Description

Pin 8 and 9 can be used as TX line and RX line for serial communications. Serial.begin configure pins for serial and set speed.

Since
Beta

Syntax 

Serial.begin(speed);
speed
speed can be set between 100 and 115200 bauds
Output
none

Example 

void setup()
{
Serial.begin(9600);  // set speed to 9600 bauds
}
 
void loop()
{
Serial.send("PINGUINO 18F2550");  // send string to the TX line
while (1);  // stop program
}

Bugs

Before 9.2, this function probably does not work for any speed except 9600
